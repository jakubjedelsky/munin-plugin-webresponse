munin plugin: webtime
=====================

USAGE
-----
`webtime [config] [autoconfig]`

DESCRIPTION
-----------
A simple Munin plugin to graph a number of seconds of webpage response.

You need to set url in plugin-conf.d. For `google.com` it can be:

  [webtime]
  env.url google.com
